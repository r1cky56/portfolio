Temperature Monitoring System using Raspberry Pi

Project Overview

This project is a Temperature Monitoring System using a Raspberry Pi, which measures and displays temperature data using a DHT11 or DHT22 sensor. The collected data can be logged and visualized on a web interface, making it a useful project for weather monitoring and IoT applications.

Apparatus Required

  • Raspberry Pi (any model with GPIO support)
  • DHT11 or DHT22 Temperature & Humidity Sensor
  • Resistors (10KΩ)
  • Jumper Wires
  • Breadboard
  • Power Supply for Raspberry Pi
  • Python installed on Raspberry Pi

How It Works

The Raspberry Pi reads temperature and humidity data from the DHT sensor and processes it using Python scripts. The working process is as follows:

  1. Sensor Initialization: The DHT sensor is connected to the Raspberry Pi’s GPIO pins.
  2. Data Collection: The Raspberry Pi reads temperature and humidity data at regular intervals.
  3. Data Processing: The collected data is processed and formatted.
  4. Output Display: The temperature and humidity data can be displayed on an LCD screen or a web interface.
  5. Data Logging: Optionally, the readings can be stored in a database for further analysis.
