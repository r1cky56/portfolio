Morse Code Converter using Arduino

Project Overview

This project is a Morse Code Converter using an Arduino, which converts text input into Morse code and outputs it through an LED or buzzer. Morse code is a method of encoding text characters using sequences of short and long signals, represented by dots and dashes. This project helps in understanding how microcontrollers process signals and can be used for communication in constrained environments where text-based transmission is not possible.

How It Works

The Arduino reads input from a user (either through a push button or serial input) and converts it into Morse code. The Morse code is then represented using LED flashes or buzzer beeps. The basic working mechanism involves the following steps:

  1. Input Processing: The user inputs a text string via the serial monitor or presses the push button to manually generate Morse code.
  2. Morse Code Encoding: The Arduino processes the input and converts each character into its respective Morse code equivalent using a predefined lookup table.
  3. Output Generation: The Morse code is transmitted as a series of short (dot) and long (dash) signals through an LED (blinking light) or a buzzer (sound pulses).
  4. Timing Control: The Arduino ensures proper spacing between dots, dashes, and letters to maintain readability according to Morse code timing rules.
  5. Real-time Communication: If a push button is used, the user can manually generate Morse code messages by pressing and holding the button for different durations.
